关于SDK

SDK概述

工银溯藏前端SDK是为银行APP和Web应用提供的区块链藏品数字身份溯源系统的前端开发工具包。

  • 提供完整的用户认证功能
  • 支持藏品的创建、查询、认领和转移
  • 提供品牌管理功能
  • 支持真伪验证和流转历史查询
  • 兼容浏览器和Node.js环境

使用说明

// 浏览器环境下使用SDK <script src="/frontend/config.js"></script> <script src="/frontend/utils.js"></script> <script src="/frontend/request.js"></script> <script src="/frontend/auth.js"></script> <script src="/frontend/collectible.js"></script> <script src="/frontend/brand.js"></script> <script src="/frontend/index.js"></script> // 使用示例 window.sdk.auth.login({ email: 'test@example.com', password: '123456' }) .then(userInfo => { console.log('登录成功:', userInfo); }) .catch(error => { console.error('登录失败:', error); });

SDK模块

模块名称 文件 主要功能
配置模块 config.js 配置API地址、超时等
请求模块 request.js 处理API请求
认证模块 auth.js 用户认证相关功能
藏品模块 collectible.js 藏品管理相关功能
品牌模块 brand.js 品牌管理相关功能
工具模块 utils.js 通用工具函数
主入口 index.js SDK主入口和模块整合

注意事项

1. 环境要求：

  • 浏览器：支持现代浏览器（Chrome、Firefox、Safari、Edge等）
  • Node.js：推荐v12.0+
  • 后端服务：需要正确配置并启动API服务和区块链网络

2. 安全提示：

  • 不要在前端代码中暴露敏感信息
  • 使用HTTPS协议进行生产环境部署
  • 定期更新JWT令牌

3. 调试建议：

  • 使用浏览器控制台查看请求和响应
  • 检查网络连接和API服务状态
  • 参考API文档确认请求参数格式
